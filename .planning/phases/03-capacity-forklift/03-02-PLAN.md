  <action>
    **ForkliftController.js** -- New file `src/interaction/ForkliftController.js`:
    - `export class ForkliftController`
    - Constructor takes `(elementManager)`:
      - `this.elementManager = elementManager`
      - `this.forklift = null` (set dynamically -- finds first forklift in elements)
      - `this.keys = { ArrowUp: false, ArrowDown: false, ArrowLeft: false, ArrowRight: false }`
      - `this.speed = 192` (pixels/sec = 4 feet/sec at 48px/ft)
      - `this.active = false` (only active when a forklift element exists)
      - Call `this.setupKeyListeners()`

    - `setupKeyListeners()`:
      - `window.addEventListener('keydown', (e) => { ... })`:
        - If `e.key in this.keys` AND `!this.isTextInputActive(e.target)`:
          - `e.preventDefault()` (prevent page scroll)
          - `this.keys[e.key] = true`
      - `window.addEventListener('keyup', (e) => { ... })`:
        - If `e.key in this.keys`: `this.keys[e.key] = false`
      - Include `isTextInputActive(target)` method (same logic as KeyboardInput.js: check tagName input/textarea/contentEditable)

    - `update(deltaTime)`:
      - Find forklift: `this.forklift = this.elementManager.getAll().find(el => el.type === 'forklift') || null`
      - If no forklift, set `this.active = false`, return
      - `this.active = true`
      - Calculate deltaSeconds = deltaTime / 1000
      - Calculate moveDistance = this.speed * deltaSeconds
      - Calculate dx, dy from key states (ArrowUp = -dy, ArrowDown = +dy, ArrowLeft = -dx, ArrowRight = +dx)
      - **Normalize diagonal movement:** If both dx !== 0 AND dy !== 0, compute magnitude = Math.sqrt(dx*dx + dy*dy), then dx = (dx/magnitude) * moveDistance, dy = (dy/magnitude) * moveDistance
      - If dx === 0 AND dy === 0: clear blocked state (`this.forklift.isBlocked = false; this.forklift.blockedElement = null`), return
      - Call `this.tryMove(dx, dy)`

    - `tryMove(dx, dy)`:
      - **Test X and Y separately** to avoid sticky collisions:
      - Save oldX = this.forklift.x
      - Apply X: `this.forklift.x += dx`
      - Check collision: `const xCollision = this.checkCollision()`
      - If xCollision: revert `this.forklift.x = oldX`
      - Save oldY = this.forklift.y
      - Apply Y: `this.forklift.y += dy`
      - Check collision: `const yCollision = this.checkCollision()`
      - If yCollision: revert `this.forklift.y = oldY`
      - Set blocked state: `this.forklift.isBlocked = !!(xCollision || yCollision)`
      - Set blocked element: `this.forklift.blockedElement = xCollision || yCollision || null`

    - `checkCollision()`:
      - Get forklift bounds: `const fb = this.forklift.getBounds()`
      - Iterate `this.elementManager.getAll()`:
        - Skip if element === this.forklift
        - Skip if element.type === 'pallet' (forklift drives over pallets)
        - Get element bounds: `const eb = element.getBounds()`
        - AABB check: `if (fb.x < eb.x + eb.width && fb.x + fb.width > eb.x && fb.y < eb.y + eb.height && fb.y + fb.height > eb.y)` return element
      - Return null (no collision)

    **Forklift.js** -- Update `src/shapes/Forklift.js`:
    - Add properties in constructor: `this.isBlocked = false`, `this.blockedElement = null`
    - Update `draw()` method:
      - Use `const baseColor = '#e8c84a'` and `const blockedColor = '#ff4444'`
      - Set `ctx.fillStyle = this.isBlocked ? blockedColor : (this.selected ? '#c8a82a' : baseColor)`
      - Set `ctx.strokeStyle = this.isBlocked ? '#cc0000' : (this.selected ? '#98790e' : '#b8992e')`
      - Set `ctx.lineWidth = this.isBlocked ? 3 / viewport.scale : 2 / viewport.scale`
      - After drawing forklift rect + label, if `this.isBlocked && this.blockedElement`:
        - Draw red outline around blocked element:
        - `const bounds = this.blockedElement.getBounds()`
        - `ctx.strokeStyle = 'rgba(255, 0, 0, 0.5)'`
        - `ctx.lineWidth = 4 / viewport.scale`
        - `ctx.strokeRect(bounds.x, bounds.y, bounds.width, bounds.height)`
