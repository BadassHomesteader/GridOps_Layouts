  <files>src/managers/ElementManager.js, src/managers/CapacityManager.js</files>
  <action>
    **ElementManager.js** -- Add observer/notify pattern:
    - Add `this.observers = []` in constructor
    - Add `subscribe(callback)` method that pushes callback to observers array
    - Add `notify(event, element)` method that calls each observer with (event, element)
    - In `add(element)`: after `this.elements.push(element)`, call `this.notify('add', element)`
    - In `remove(element)`: after `this.elements.splice(index, 1)`, call `this.notify('remove', element)`
    - In `clear()`: after `this.elements = []`, call `this.notify('clear', null)`

    **CapacityManager.js** -- New file `src/managers/CapacityManager.js`:
    - `export class CapacityManager`
    - Constructor takes `elementManager` parameter
    - Properties: `this.elementManager`, `this.observers = []`, `this.totalCapacity = 0`, `this.ceilingHeight = 144` (12 feet in inches, matching 48px=1ft scale where palletHeight is in inches)
    - `subscribe(callback)` -- push to observers
    - `notify()` -- call each observer with `this.totalCapacity`
    - `recalculate()` -- iterate `this.elementManager.getAll()`:
      - If `element.type === 'rack'`: add `element.totalCapacity`
      - If `element.type === 'pallet'`: add `Math.max(1, Math.floor(this.ceilingHeight / element.palletHeight))`
      - Set `this.totalCapacity = total`, call `this.notify()`
    - `setCeilingHeight(inches)` -- set `this.ceilingHeight = inches`, call `this.recalculate()`
    - `getCeilingHeight()` -- return `this.ceilingHeight`
    - `getTotal()` -- return `this.totalCapacity`
    - In constructor: subscribe to elementManager changes: `this.elementManager.subscribe(() => this.recalculate())` and call initial `this.recalculate()`
  </action>
  <verify>

  </verify>
  <done>ElementManager notifies observers on add/remove/clear. CapacityManager auto-recalculates on element changes and notifies its own subscribers with updated total.</done>
