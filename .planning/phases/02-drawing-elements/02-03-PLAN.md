**src/interaction/DragMove.js -- DragMoveController class:**
- Constructor takes `(selectionManager, coordinateConverter, grid)`
- State: `isDragging (false)`, `dragOffset ({x: 0, y: 0})` -- offset from element origin to click point in world coords
- `startDrag(worldX, worldY)`:
  - Get selected element from selectionManager.getSelected()
  - If no selection, return false
  - Calculate offset: `dragOffset.x = worldX - element.x`, `dragOffset.y = worldY - element.y`
  - Set isDragging = true
  - Return true (indicates drag started)
- `updateDrag(worldX, worldY, shiftKey)`:
  - If not isDragging, return
  - Get selected element
  - Calculate new position: `newX = worldX - dragOffset.x`, `newY = worldY - dragOffset.y`
  - If !shiftKey (snap mode): snap newX, newY using grid.snapToGrid()
  - Set element.x = snapped.x, element.y = snapped.y
- `endDrag()`:
  - Set isDragging = false
  - Reset dragOffset
- `getIsDragging()`: return isDragging

**src/main.js mouse event refactor:**

The mousedown/mousemove/mouseup handlers need a 3-way priority system:
1. Element drag (if clicking on a selected element)
2. Element select (if clicking on an unselected element)
3. Pan (if clicking on empty canvas)

Refactor the mousedown handler:
```javascript
canvas.addEventListener('mousedown', (event) => {
  const canvasX = event.clientX - canvasRect.left;
  const canvasY = event.clientY - canvasRect.top;
  const world = coordinateConverter.screenToWorld(canvasX, canvasY);

  // Priority 1: Check if clicking on currently selected element (start drag-move)
  const selected = selectionManager.getSelected();
  if (selected && selected.containsPoint(world.x, world.y)) {
    dragMoveController.startDrag(world.x, world.y);
    canvas.style.cursor = 'move';
    return;
  }

  // Priority 2: Check if clicking on any element (select it)
  const hitElement = elementManager.getElementAtPoint(world.x, world.y);
  if (hitElement) {
    selectionManager.selectElement(hitElement);
    // Also start drag immediately (select + drag in one motion)
    dragMoveController.startDrag(world.x, world.y);
    canvas.style.cursor = 'move';
    return;
  }

  // Priority 3: Empty canvas -- clear selection and start pan
  selectionManager.clearSelection();
  isPanning = true;
  lastX = event.clientX;
  lastY = event.clientY;
  canvas.classList.add('panning');
});
```

Refactor the mousemove handler:
```javascript
canvas.addEventListener('mousemove', (event) => {
  const canvasX = event.clientX - canvasRect.left;
  const canvasY = event.clientY - canvasRect.top;

  // Element drag takes priority
  if (dragMoveController.getIsDragging()) {
    const world = coordinateConverter.screenToWorld(canvasX, canvasY);
    dragMoveController.updateDrag(world.x, world.y, event.shiftKey);
    return;
  }

  // Pan handling
  if (isPanning) {
    const deltaX = event.clientX - lastX;
    const deltaY = event.clientY - lastY;
    viewport.pan(deltaX, deltaY);
    lastX = event.clientX;
    lastY = event.clientY;
    return;
  }

  // Hover: update cursor and snap indicator
  currentMouseWorld = coordinateConverter.screenToWorld(canvasX, canvasY);
  const hoverElement = elementManager.getElementAtPoint(currentMouseWorld.x, currentMouseWorld.y);
  canvas.style.cursor = hoverElement ? 'pointer' : 'default';
});
```

Refactor the mouseup handler:
```javascript
canvas.addEventListener('mouseup', () => {
  if (dragMoveController.getIsDragging()) {
    dragMoveController.endDrag();
    canvas.style.cursor = 'default';
    return;
  }
  isPanning = false;
  canvas.classList.remove('panning');
});
```

Also update mouseleave to end both drag and pan.
